1 自动版本号

昨天 2016.03.02 刚发现的好东西，更优雅的 Android 发布自动版本号方案

这里简单总结下，配合 git 获取软件版本号和版本名

版本号 versionCode
使用 Git 中 commit 的数量来作为版本号——versionCode

版本名 versionName
使用 git describe，获取从当期 commit 到距离它最近的 tag 的描述。默认都是 annoted tag，如果要指所有的类型的 tag 的话，就加 –tags 参数。
